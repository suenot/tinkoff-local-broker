# Пример бэктеста робота на JavaScript

1. Запустить tinkoff-local-broker
```
docker run --init --rm -p 8080:8080 -v $(pwd)/.cache:/app/.cache vitalets/tinkoff-local-broker
```

2. Запустить бэктест
```
npx ts-node-esm examples/js/backtest.ts
```
